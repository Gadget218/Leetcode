from collections import Counter
class Solution:
    def uniqueOccurrences(self, arr: List[int]) -> bool:
        h1=Counter(arr)
        h2=Counter(h1.values())
        for i in h2.values():
            if i>1:
                return False
        return True
        
Time: O(n)

Operation		Cost
Counter(arr)		O(n)
Counter(h1.values())	O(k) where k = unique elements ≤ n
loop over h2.values()	O(k)
Total			O(n)

Space: O(k) where k = number of unique elements in arr

Storage		Size
h1		O(k)
h2		O(k)
In the worst case all elements are unique so k = n, making it O(n).
