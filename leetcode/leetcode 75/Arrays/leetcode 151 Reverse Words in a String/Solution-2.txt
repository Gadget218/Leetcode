class Solution:
    def reverseWords(self, s: str) -> str:
        s1 = s.strip()
        l = s1.split(' ')   # keep your structure
        op = []

        for i in range(len(l)-1, -1, -1):
            if l[i] != "":   # remove empty strings
                op.append(l[i])

        return " ".join(op)

Explanation:
split() without arguments automatically removes extra spaces.

class Solution:
    def reverseWords(self, s: str) -> str:
        l = s.split()
        return " ".join(l[::-1])

Why this works
"  hello   world  ".split()


Result:

['hello', 'world']

Complexity

Time: O(n)

Space: O(n)
