class Solution:
    def maxOperations(self, nums: List[int], k: int) -> int:
        nums.sort()
        i, j = 0, len(nums) - 1
        op = 0
        
        while i < j:
            s = nums[i] + nums[j]
            
            if s == k:
                op += 1
                i += 1
                j -= 1
            elif s < k:
                i += 1
            else:
                j -= 1
        
        return op

Time Complexity:

Sorting: O(n log n)

Two-pointer scan: O(n)

Total:

O(n log n)

Space Complexity

Sorting in Python (Timsort): O(n) (not strictly in-place)

Subtle Point (Advanced)

Two-pointer can be O(1) space in languages where sorting is in-place (like C++)

But in Python → sorting uses extra memory → O(n)
